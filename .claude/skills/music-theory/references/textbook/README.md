# Textbook Reference — Open Music Theory (Stripped)

This directory is a pointer to the full stripped textbook for deep
fallback. **You should rarely need this.** The other reference files in
`../` cover the essential concepts. Only come here when you encounter
something unusual not covered by those files.

## Location

The full distilled textbook lives at:
```
docs/references/openmusictheory.github.io/open_music_theory_distilled.md
```

It covers (in order):
1. **Intervals** — chromatic, diatonic, compound, inversion, consonance/dissonance
2. **Triads and seventh chords** — qualities, Roman numerals, lead-sheet symbols
3. **Harmonic functions** — T/S/D, functional triggers/associates/dissonances
4. **Harmonic syntax** — the idealized phrase, T→S→D→T cycle, prolongation
5. **Counterpoint** — cantus firmus, species 1–4 (voice-leading foundations)
6. **Galant schemata** — Meyer, Jupiter, Prinner, Fonte, Monte, Ponte
7. **Classical themes** — sentence, period, hybrid types
8. **Sonata form** — exposition, development, recapitulation (rarely relevant)
9. **Modal mixture** — borrowing from parallel keys
10. **Applied chords** — secondary dominants, tonicization
11. **Modulation** — direct, pivot-chord, step-up

## When to read which section

| Your question | Read section |
|--------------|-------------|
